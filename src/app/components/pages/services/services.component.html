With todays technology growing at such a fast pace, websites have to keep up with all the different devices that can browse the web. Responsive design is a giant leap forward for allowing all these devices to view appropriate data on our sites. The problem is sometimes the data we would show on a device like a computer would and should have more information then what you want to show a person on a device like a phone. With this plugin we trim down paragraphs for mobile users allowing for a better viewing while still allowing them access to all the information. Reduce Text

Works per paragraph

This plugin works on per paragraph basis so each one can be expanded and contracted for maxium mobile friendliness. =) Reduce Text

Works on short & long paragraphs

No matter how long or short your paragraph is, you can add easily add or remove the plugin to suite your needs. Don't want the plugin to shorten your text? Don't add the recude link and it wont!
